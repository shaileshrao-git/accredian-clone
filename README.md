# Accredian Enterprise Page Clone

This project is a partial clone of the Accredian Enterprise landing page built using Next.js as part of the Full Stack Developer Intern assignment.

## ⚙️ Setup Instructions

1. Clone the repository:
git clone <your-repo-link>  
cd accredian-clone  

2. Install dependencies:
npm install  

3. Run the project:
npm run dev  

4. Open in browser:
http://localhost:3000  

## 🧠 Approach Taken

I began by analyzing the reference website and breaking it down into reusable components such as Hero, Navbar, Stats, Domain Expertise, FAQ, Testimonials, and Footer.

The project was built using Next.js App Router for structure and Tailwind CSS for styling and responsiveness. I followed a component-based approach to keep the code clean and maintainable.

Each section was developed individually to match the layout and design of the reference website. I ensured that the UI is fully responsive by using mobile screenshots as a guide and applying a mobile-first design approach.

Interactive features like the enquiry modal popup, FAQ accordion, and testimonials slider were implemented to enhance user experience. I focused on maintaining proper spacing, alignment, and consistent styling across the application.
